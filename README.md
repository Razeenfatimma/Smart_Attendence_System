# Smart Attendance System
# Status:  Still in Development (Active Work in Progress)

A Java-based desktop application designed to streamline the process of tracking student attendance using a modern, user-friendly interface.

# Features (In Development)
Modern Login UI: A login interface.

Object-Oriented Design: Built using core Java principles including Inheritance, Polymorphism, and Encapsulation.

Course Management: Ability to manage different courses and lectures.

Swing GUI: Developed using Java Swing components and customized Layout Managers.

# Built With
Java: Core programming logic.

Java Swing: Graphical User Interface (GUI).

NetBeans IDE: Development environment.

SQL (Planned): For database management and data persistence
